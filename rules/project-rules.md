# starter-fullstack — project-specific правила

- **Tailwind CSS 4** — через Vite plugin, не CLI / не PostCSS.
- **`entrypoint.sh`** — не менять без понимания порядка запуска nginx/uvicorn.
- **In-memory store** в backend — пример, заменить на БД при клонировании.
- **Шаблон публичный** — не содержит секретов и prod-конфигов.
- **Не снижать порог coverage** ниже 100%.
